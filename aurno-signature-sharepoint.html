How to Install Your Signature


        

          
1

          

            
Apple Mail

            
Click Copy HTML Signature. In Mail go to Settings → Signatures → +. Open TextEdit in plain-text mode, paste, select all, copy, then paste into the Mail signature field.

          

        

        


        

          
2

          

            
Outlook (Mac & Windows)

            
Go to New Email → Insert → Signature → Edit Signatures. Click the </> source button in the editor, paste the HTML code, then save.

          

        

        


        

          
3

          

            
Gmail

            
Paste the HTML into a new Compose window body, select all, copy. Then go to Settings → See all settings → Signature and paste into the signature field.
