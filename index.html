How to Install Your Signature


        

          
1

          

            
Outlook Web (Recommended — Image method)

            
Click Copy as Image, then go to Settings → Mail → Compose and reply → Signature. Click in the signature box and paste with Cmd+V. Done.

          

        

        


        

          
2

          

            
Apple Mail

            
Click Copy as Image, go to Mail → Settings → Signatures → +, click in the signature area and paste with Cmd+V.

          

        

        


        

          
3

          

            
Gmail

            
Click Copy as Image, go to Settings → See all settings → Signature, click in the box and paste with Cmd+V.

          

        

        


        

          
4

          

            
HTML method (advanced)

            
Use Copy HTML Code for clients that support HTML source editing. Links will be clickable. The image method is easier for most people.
